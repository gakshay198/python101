********** Checklist whenever you are using a function **********

1. Did you start your function definition with def?
2. Does your function name have only characters and _ characters?
3. Did you put an open parenthesis right after the function name?
4. Did you put an open parenthesis right after the function name?
5. Did you make each argument unique (no duplicate names)?
6. Did you put a close parenthesis and a colon after the arguments?
7. Did you indent all lines of code you want in the function exactly
    four spaces?
8. Did you end your function by going back to writing with no indent?

When you run (“use” or “call”) a function, check these things:
1. Did you call	this function by typing	its	name?
2. Did you put the ( character after the name to run it?
3. Did you put the values you want into	the	parentheses	separated
    by commas?
4. Did	you	end	the function call with a ) character?
